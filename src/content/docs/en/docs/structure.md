## Directory Overview

- `src/`: Contains the source code for the website.
  - `components/`: Reusable Astro and Vue components.
  - `layouts/`: Page layouts used across the site.
  - `pages/`: Individual pages of the website.
  - `content/`: Markdown and MDX content for the blog and documentation.
  - `styles/`: Global styles and CSS configurations.
- `public/`: Static assets like images, fonts, and robots.txt.
- `astro.config.ts`: Configuration file for the Astro framework.
- `package.json`: Project dependencies and scripts.

## Tech Stack

- **Framework**: [Astro](https://astro.build/)
- **UI Library**: [Rimelight UI](https://rimelight.com)
- **Styling**: UnoCSS / Vanilla CSS
- **Documentation**: [Starlight](https://starlight.astro.build/)
